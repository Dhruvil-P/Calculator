# Calculator
It's a standard calculator made in Flutter.
## Description
### It performs operations like: 
* Division 
* Multiplication
* Addition
* Subtraction
* Percentage
### Moreover, there are features like: 
* Adding decimal point
* Adding/Removing negative sign
* Clearing the display bar
